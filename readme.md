# Monster Jam Api

> Backend API for the Monster Jam application which is a academy directory website

## Usage

rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
